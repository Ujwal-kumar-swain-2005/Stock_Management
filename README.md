# Inventory Management System

A full-stack automated Inventory and Stock Management System built with Spring Boot and React.

## Tech Stack

### Backend
- Java 17
- Spring Boot 3.2
- Spring Data JPA
- Spring Security + JWT
- MySQL 8

### Frontend
- React 18
- React Router
- Axios
- Recharts
- Bootstrap 5

### DevOps
- Docker & Docker Compose

## Getting Started

### Prerequisites
- Java 17+
- Node.js 18+
- MySQL 8+
- Docker (optional)

### Backend Setup
```bash
cd backend
mvn clean install
mvn spring-boot:run
```

### Frontend Setup
```bash
cd frontend
npm install
npm start
```

### Docker Setup
```bash
docker-compose up --build
```
